n文字制限メモ - クラウド同期対応

⚠️ オフラインモード - ローカルのみで動作中

入力制限設定

メモ

0 文字 / 0 バイト
制限: 0 / 200 文字 (残り 200)

暗号化（オプション）

現在: 平文（暗号化なし）

操作ヒント
  • パスワード欄で Enter を押すと復号を試みます。
  • 復号済みの状態でパスワード欄を空にして「暗号化して保存」を押すと、暗号を解除して平文として保存できます。
  • パスワードを変更する場合は、先に復号してから新しいパスワードを入力して「暗号化して保存」を押してください。

暗号化に使うパスワードは端末にもGistにも保存されません。忘れると復号できません。

クラウド同期

参考: gist の内容は https://gist.github.com/ の"View your gists"で確認できます。ファイル名は memo.json で、description は変更しないでください。

使い方

  • ローカル保存: 入力内容は自動的に端末に保存されます
  • クラウド同期: GitHub でログインすると、複数端末でメモを共有できます
  • 暗号化（任意）: パスワードでメモ本文を暗号化してから同期できます（鍵は保存されません）
  • オフライン対応: ネットワーク接続がなくても使用できます
  • PWA対応: ホーム画面に追加してアプリのように使用できます

認証（Firebase + GitHub）

このアプリは Firebase Authentication を通じて GitHub にサインインします。 「GitHub でログイン」ボタンを押すと、GitHub の認可画面がポップアップで開きます。

  1. 画面の「GitHub でログイン」ボタンをクリック
  2. 表示される GitHub の認可画面でアクセス許可（gist スコープ）を承認
  3. 承認が完了するとポップアップが閉じ、アプリに戻ります
  4. 自動でトークンが取得され、同期が可能になります
