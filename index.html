クラウド同期

使い方

  • ローカル保存: 入力内容は自動的に端末に保存されます
  • クラウド同期: GitHub でログインすると、複数端末でメモを共有できます
  • オフライン対応: ネットワーク接続がなくても使用できます
  • PWA対応: ホーム画面に追加してアプリのように使用できます

GitHub Personal Access Token の作成方法

  1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. 「Generate new token (classic)」をクリック
  3. Note: 任意の名前（例: Memo App）
  4. Expiration: 任意の期限
  5. 「gist」にチェックを入れる
  6. 「Generate token」をクリック
  7. 生成されたトークンをコピーしてログイン時に入力

⚠️ トークンは安全に管理してください。他人に共有しないでください。
