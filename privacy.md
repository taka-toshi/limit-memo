# プライバシーポリシー

最終更新日: 2026年2月26日

このドキュメントは、Memo App（以下「当アプリ」）が取得・保存・利用する情報と、その取り扱い方針を示します。

## 1. 要点
- ユーザーが「GitHubでログイン」した場合、Firebase Authentication を通じて GitHub に登録されたメールアドレスが当アプリに渡されます。開発者（サービス提供者）はそのメールアドレスを確認できます。
- GitHubログインを行わない場合、当アプリ側で個人を識別できる情報は収集・送信しません。メモは端末の `localStorage` にのみ保存されます。

## 2. 取得する情報
### 2.1 GitHubログイン時に取得する情報
- GitHub に紐づくメールアドレス（Firebase が提供）
- Firebase Authentication のユーザー識別子（UID）
- GitHub OAuth のアクセストークン（クライアント側に一時保存されることがあります）
- 最初のログイン日
- 最終ログイン日

### 2.2 GitHubログインを行わない場合
- 収集する個人情報はありません。メモは端末の `localStorage` にのみ保存され、外部に送信されません。

### 2.3 収集しない情報
- GitHubのユーザー名
- GitHubのプロフィール画像
- Gist以外のリポジトリ内容
- その他GitHubの個人情報（フォロー情報、スター情報等）
- メモの内容（ユーザーが同期を有効にした場合、メモ内容はユーザー自身の GitHub Gist に保存されますが、当アプリのサーバーは介在しません）
- アプリ利用状況を外部に送信するトラッキング（Analytics、広告等）
- Cookie

## 3. 利用目的
- 認証（ログインのため）
- GitHub Gist を用いたユーザー自身のメモの同期

## 5. 開発者がメールアドレスを確認できる場合について
Firebase コンソールやアプリ側の管理画面・ログから、ユーザーのメールアドレスを確認できる場合があります。これは主にアカウント識別および同期機能のためであり、目的外利用や第三者提供は行いません。

## 6. データの削除・退会方法
- クラウド上のメモ（Gist）はユーザー自身が GitHub 側で削除できます。
- ローカルのデータはブラウザのストレージを削除するか、アプリの「ローカルデータ削除」操作で消去してください。
- アカウント削除を希望する場合は、アプリ内の「アカウント削除」機能を利用してください。これにより、Firebase Authentication のユーザーデータも削除されます。

## 7. データの保持期間

- 認証情報: ユーザーがアカウント削除を要求するまで、または、サービスが終了するまで

## 8. セキュリティ

当アプリは以下のセキュリティ対策を実施しています：

- HTTPS通信の使用
- Firebase Authenticationによる認証
- GitHub OAuth による安全なログイン

ただし、以下の点にご注意ください：

- メモデータは暗号化されていません
- 共有PCでの利用は推奨しません

## 9. 子どものプライバシー

13歳未満の方の利用を想定していません。（github.comの利用規約により、13歳未満の方はGitHubアカウントを作成できません）
13歳未満の方が誤って登録した場合、速やかにご連絡ください。

## 10. プライバシーポリシーの変更

本ポリシーは予告なく変更される場合があります。
重要な変更がある場合、アプリ内で通知します。
